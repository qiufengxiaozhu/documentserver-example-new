# Integration examples
These examples are simple document management systems. Files in these systems can use FilezOffice to view online or to coedit online. They show the way to integrate FilezOffice into your own website(application) using one of the integration solutions. Do NOT use these examples in your production without any modification.

If you prefer to implement server callback API to implement the integration, please refer this example
[server call back integrtion solution](./server-callback-solution/java/filez-demo/)

If you prefer to pass information in url to implement the integration, please refer this example
[pass information in url solution](./params-in-url-solution/java/filez-demo/)
